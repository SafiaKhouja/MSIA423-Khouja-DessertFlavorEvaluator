About the Dessert Flavor Evaluator

Flavor combination ratings are predicted using a gradient boosted decision tree model (XGBoost). The test R^2 of the model is 60%.

The model was trained on a dataset of over 6500 Epicurious dessert recipes. The dataset is static so the ratings given by the recommendation system might not prefectly reflect the current ratings listed on Epicurious.com.

The recommendation system identifies popular and highly-rated recipes that fit the flavor combination. Recipes with less than 6 reviews will not be recommended, because the sample size is too small.
